# Ashish Chauhan – Personal Portfolio Website

A modern, responsive personal portfolio website showcasing my skills, projects, certifications, and resume as a tech student.

Built with a clean UI and a recruiter-focused layout..

---

## Live Website
https://ashish4chauhan.github.io/ashish-portfolio/

---

## Features
- Modern and professional design
- Fully responsive (mobile & desktop)
- About Me section with justified text
- Skills, Projects, and Certificates sections
- Resume download button
- Contact section with icons
- Smooth scrolling navigation
- Lightweight and fast (no frameworks)

---

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript
- Font Awesome (icons)
- Google Fonts (Inter)
